# Exercise Tracker Microservice

### APIs and Microservices Projects for FCC

GET user's exercise log:

    GET /api/users/:_id/logs?[from][&to][&limit]

**[ ]** = optional

**from, to** = dates (yyyy-mm-dd); **limit** = number

The project available on replit.com
